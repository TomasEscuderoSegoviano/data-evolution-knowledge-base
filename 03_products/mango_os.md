# Mango OS / Radix IoT

## Objetivo

Documentar la propuesta, narrativa, capacidades, beneficios y mensajes clave de Mango OS / Radix IoT dentro del portafolio de Data Evolution.

Este archivo sirve como base para presentaciones comerciales, propuestas, campañas, landing pages, contenidos de LinkedIn, emails, guiones, materiales de ventas y conversaciones estratégicas sobre monitoreo, integración de datos, visualización operativa, BMS, edificios digitales e infraestructura crítica.

## Descripción general

Mango OS / Radix IoT es una plataforma orientada a monitoreo, integración de datos, visualización operativa y conectividad de sistemas para infraestructura crítica y edificios digitales.

Permite conectar información de diferentes sistemas, dispositivos y fuentes operativas para generar mayor visibilidad sobre el desempeño de instalaciones, activos y procesos tecnológicos.

## Descripción corta

Mango OS / Radix IoT permite integrar, monitorear y visualizar datos operativos de infraestructura crítica y edificios digitales.

Mango OS / Radix IoT permite consolidar datos de sistemas operativos y tecnológicos en una plataforma de monitoreo y visualización que ayuda a mejorar control, eficiencia y seguimiento de infraestructura crítica.

## Objeciones frecuentes

### Ya tenemos BMS

Mango OS / Radix IoT puede complementar sistemas existentes al centralizar datos, facilitar visualización y conectar información operativa de diferentes fuentes.

### Ya tenemos dashboards

El valor no está solo en visualizar, sino en integrar sistemas, centralizar datos y facilitar seguimiento operativo.

### Nuestros sistemas son diferentes

La integración debe evaluarse según protocolos, conectividad, alcance técnico y objetivos operativos.

### No queremos reemplazar plataformas existentes

Mango OS / Radix IoT debe posicionarse como una capa de integración y monitoreo, no necesariamente como sustituto total de sistemas existentes.

### No tenemos datos ordenados

Puede iniciarse con una evaluación de fuentes disponibles, prioridades operativas y sistemas candidatos a integración.

## Preguntas frecuentes

### ¿Qué es Mango OS / Radix IoT?

Es una plataforma orientada a integrar, monitorear y visualizar datos operativos de sistemas, dispositivos e infraestructura.

### ¿Aplica solo para edificios digitales?

No. También puede aplicarse a infraestructura crítica, centros de datos y entornos donde se requiere conectar datos de operación.

### ¿Reemplaza al BMS?

No necesariamente. Puede integrarse o complementar sistemas existentes según el alcance técnico y los objetivos del cliente.

### ¿Puede integrarse con sensores?

Sí, puede conectarse con dispositivos y sensores según compatibilidad, protocolos y validación técnica.

### ¿Qué valor aporta a servicios administrados?

Permite contar con datos operativos para monitoreo, seguimiento, mantenimiento y soporte técnico.
