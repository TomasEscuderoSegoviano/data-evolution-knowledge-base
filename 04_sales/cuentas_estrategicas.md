# Cuentas estratégicas

## Objetivo

Documentar las cuentas estratégicas, cuentas nombradas, oportunidades comerciales, contactos, responsables internos, contexto de negocio y acciones sugeridas para el seguimiento comercial de Data Evolution.

Este archivo sirve como base para ventas, marketing, campañas ABM, seguimiento de cuentas, priorización comercial, CRM, prospección dirigida y generación de materiales personalizados.

## Principio de uso

Las cuentas estratégicas deben gestionarse con enfoque consultivo y de desarrollo de cuenta, no solo como contactos aislados.

Cada cuenta debe analizarse considerando:

- Potencial comercial.
- Relación actual.
- Proyecto identificado.
- Necesidad operativa.
- Soluciones relacionadas.
- Responsable interno.
- Próxima acción.
- Nivel de prioridad.
- Estado en CRM.
- Materiales requeridos.
- Riesgos o bloqueos.
- Posible enfoque ABM.
- Histórico de relación.
- Contactos activos o pendientes.

## Criterios para clasificar cuentas estratégicas

Una cuenta puede considerarse estratégica si cumple uno o varios criterios:

- Tiene proyectos activos o potenciales de infraestructura crítica.
- Opera centros de datos o infraestructura tecnológica de alta disponibilidad.
- Tiene presencia regional o expansión.
- Requiere servicios especializados de consultoría, ingeniería o mantenimiento.
- Puede abrir oportunidades en nuevos mercados.
- Tiene relación con fabricantes, integradores o aliados clave.
- Tiene potencial para soluciones digitales, IA, monitoreo o eficiencia energética.
- Puede convertirse en caso de éxito o referencia comercial.
- Requiere seguimiento ejecutivo o técnico especializado.
- Pertenece a una lista de cuentas nombradas para seguimiento comercial.
- Tiene asignación interna dentro del equipo comercial.

# 6. Recomendaciones para seguimiento

- No enviar materiales genéricos sin contexto.
- Registrar todos los contactos en CRM.
- Definir responsable interno por cuenta.
- Asociar cada cuenta con una línea de solución prioritaria.
- Documentar siguiente acción y fecha de seguimiento.
- Identificar si la cuenta requiere enfoque ejecutivo, técnico o comercial.
- Usar mensajes por cargo para personalizar acercamientos.
- Usar manual de servicios solo cuando la necesidad esté clara.
- Priorizar cuentas con proyectos activos o potencial de expansión.
- Separar cuentas activas, cuentas por desarrollar y cuentas por investigar.
- No incluir cuotas ni montos en documentos de trabajo para LLMs.
- Validar información sensible antes de cargarla a cualquier sistema externo.
- Mantener actualizado el estado de asignación por cuenta.
- Para cuentas sin asignación, definir owner antes de activar campañas.
- Para cuentas con caso de éxito, validar uso público antes de publicar logos o menciones.
