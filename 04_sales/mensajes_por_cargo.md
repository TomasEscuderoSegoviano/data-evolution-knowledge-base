# Mensajes por cargo

## Objetivo

Documentar mensajes comerciales adaptados por cargo para apoyar campañas, prospección, correos, LinkedIn, presentaciones comerciales, landing pages, materiales de ventas y estrategias ABM de Data Evolution.

Este archivo complementa `04_sales/buyer_personas.md` y permite traducir la propuesta de valor de Data Evolution según las prioridades, dolores y lenguaje de cada interlocutor.

## Principio de uso

El mensaje debe adaptarse al cargo.  
No todos los perfiles compran por las mismas razones.

- Dirección compra visión, riesgo, continuidad y crecimiento.
- Tecnología compra escalabilidad, integración y control.
- Operaciones compra continuidad, eficiencia y reducción de fricción.
- Ingeniería compra viabilidad, estándares y precisión técnica.
- Facility compra mantenimiento, condiciones físicas y trazabilidad.
- Finanzas compra justificación, riesgo y priorización de inversión.
- Compras compra alcance, cumplimiento y comparabilidad.

## Mensaje base transversal

Data Evolution integra infraestructura crítica, datos, inteligencia artificial y servicios especializados para ayudar a las organizaciones a operar entornos tecnológicos con mayor visibilidad, eficiencia y continuidad.

## Recomendaciones de uso

- Usar este archivo para adaptar mensajes por cargo antes de crear campañas o correos.
- No enviar mensajes técnicos extensos a perfiles ejecutivos.
- No enviar mensajes genéricos a perfiles técnicos.
- Conectar siempre el mensaje con una necesidad del cargo.
- Evitar vender productos aislados sin contexto operativo.
- Usar CTAs consultivos, no agresivos.
- Para cuentas estratégicas, combinar cargo + industria + etapa del ciclo de vida.
- Para campañas ABM, crear versiones específicas por cargo y vertical.
