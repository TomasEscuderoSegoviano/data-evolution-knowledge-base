# Data Evolution Knowledge Base

Base de conocimiento estructurada para documentar, reutilizar y activar información estratégica, comercial, técnica y editorial de Data Evolution en diferentes modelos de lenguaje e interfaces de inteligencia artificial.

## Objetivo

Centralizar el conocimiento de Data Evolution en una estructura modular, portable y actualizable, diseñada para ser utilizada con cualquier LLM, sistema RAG, agente de IA o herramienta de automatización.

## Alcance

Esta base de conocimiento cubre:

- Identidad, tono y mensajes clave de marca.
- Perfil institucional de Data Evolution.
- Portafolio de soluciones y servicios.
- Productos y plataformas digitales.
- Campañas de marketing y contenidos.
- Procesos comerciales y seguimiento de cuentas.
- CRM, automatización, UTMs y clasificación de leads.
- Prompts reutilizables para generación de contenido y análisis.

## Principios de uso

1. La información debe mantenerse modular.
2. Cada archivo debe cubrir un solo tema principal.
3. Todo contenido debe incluir fecha de actualización.
4. Los claims técnicos y comerciales deben estar validados.
5. La base debe funcionar como fuente maestra, no como archivo final de diseño.
6. Los documentos deben escribirse en Markdown para facilitar su uso en diferentes plataformas.

## Estructura general

- `00_brand/`: voz, tono, mensajes clave y claims.
- `01_company/`: información institucional.
- `02_portfolio/`: oferta de soluciones y servicios.
- `03_products/`: productos, plataformas y tecnologías clave.
- `04_sales/`: ventas, cuentas, objeciones y mensajes comerciales.
- `05_marketing/`: campañas, contenidos, eventos y canales.
- `06_crm_automation/`: CRM, automatización, datos y seguimiento.
- `07_prompts/`: prompts reutilizables.
- `08_events/`: eventos, cursos y activaciones.
- `09_sources/`: fuentes internas, documentos base e historial.
